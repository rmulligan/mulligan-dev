# mulligan.dev

Static site built with Hugo.

## Local preview
```bash
hugo server --buildDrafts
```

## Deployment (Cloudflare Pages)
- Build command: `hugo`
- Output directory: `public`
- Branch: `main`
- Custom domain: `mulligan.dev`

## Structure
- Posts: `content/posts/`
- Static assets: `static/`
- Theme: `themes/mulligan-minimal/`
