# OSF Preprints Checklist

## Before submission
- Confirm arXiv status (endorsement pending)
- Ensure title/abstract match current results
- Confirm author list and affiliations
- Verify figures and captions
- Check licensing and sharing permissions

## Submission steps
1. Create OSF account (if needed)
2. Start a new preprint
3. Upload PDF
4. Enter metadata (title, abstract, keywords)
5. Choose subject areas
6. Add contributors
7. Submit for moderation

## After submission
- Record DOI
- Add DOI link to mulligan.dev and Medium
- Add DOI link to Hugging Face model card
